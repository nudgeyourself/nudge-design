### Overview

The tabs component is used to navigate pages of content.

---

### Props

### tabs

`Tab[]` An array of tabs.

```jsx
export interface Tab {
  id: string;
  content: React.ReactNode;
}
```

---

### tabSelected

`(id) => {}` A callback when a tab is clicked.

### highlighted

The ID of a tab as a `string` that highlights the tab, prompting the user to make that selection. This is generally only used during onboarding.

</BrowserOnly>
