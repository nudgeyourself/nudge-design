### Overview

Collects a group of buttons and creates standardized spacing between each.

---

### Props

**children**: `React.ReactNode` Inject button components to be grouped.

**row?**: `boolean` Display the group of buttons inline.

**margin?**: `boolean` Add standardized spacing around the ButtonGroup.
