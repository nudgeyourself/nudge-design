### Overview

The tabs component is used to navigate pages of content.

---

### Props

**tabs** : `Tab[]` An array of tabs.

```jsx
export interface Tab {
  id: string;
  content: React.ReactNode;
}
```

**selected**: `string` Which tab is currently selected.

**tabSelected**: `(id) => {}` Callback when a tab is clicked.

**expanded?**: `boolean` Expand the tabs to fill the container.

**margin?**: `boolean` Adds standard space around the tabs.
